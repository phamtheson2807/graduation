🚀 Triển khai dự án Gallery Tốt Nghiệp lên dataonline.vn

✅ Package đã sẵn sàng!

Tất cả file cần thiết đã được chuẩn bị trong thư mục deploy-package

📋 Checklist trước khi triển khai

  • Đăng ký hosting tại dataonline.vn
  • Có thông tin FTP/File Manager
  • Đăng ký ImgBB API (miễn phí)
  • Chuẩn bị mật khẩu admin mạnh

🔧 Bước 1: Cấu hình API upload ảnh

1.1. Đăng ký tài khoản miễn phí tại: https://imgbb.com/

1.2. Vào phần "API" và copy API key

1.3. Mở file api-config.js trong deploy-package

1.4. Tìm dòng:

IMGBB_API_KEY: 'your_imgbb_api_key_here'

1.5. Thay 'your_imgbb_api_key_here' bằng API key thật

🔒 Bước 2: Đặt mật khẩu admin

2.1. Trong file api-config.js, tìm dòng:

DEFAULT_PASSWORD: 'graduation2025'

2.2. Đổi 'graduation2025' thành mật khẩu mạnh của bạn

Ví dụ: DEFAULT_PASSWORD: 'MyStrong2025!@#'

⚠️ Lưu ý bảo mật

• Không chia sẻ mật khẩu admin
• Có thể đổi tên file admin.html thành tên khác
• Backup thường xuyên

📤 Bước 3: Upload lên hosting

3.1. Đăng nhập hosting control panel dataonline.vn

3.2. Vào File Manager hoặc dùng FTP client

3.3. Upload tất cả file trong deploy-package vào thư mục:

public_html/

3.4. Đảm bảo cấu trúc file như sau:

public_html/
├── index.html
├── gallery.html
├── admin.html
├── style.css
├── script.js
├── gallery.js
├── admin.js
├── api-config.js
├── phepmau.mp3
├── .htaccess
└── robots.txt

🌐 Bước 4: Cấu hình domain

4.1. Trong hosting control panel, cấu hình:

  • ✅ SSL Certificate (HTTPS)
  • ✅ GZIP Compression
  • ✅ CDN (nếu có)

4.2. Domain sẽ có dạng:

https://yourname.dataonline.vn

❓ Troubleshooting

Lỗi upload ảnh:

  • Kiểm tra ImgBB API key đã đúng chưa
  • File ảnh có vượt quá 5MB không
  • Mở F12 Console xem lỗi gì

Không vào được admin:

  • Kiểm tra mật khẩu đã đổi đúng chưa
  • Clear cache browser và thử lại
  • Kiểm tra file admin.html đã upload chưa

Website không hiển thị:

  • Kiểm tra DNS đã trỏ đúng chưa
  • Đợi 24h để DNS propagate
  • Liên hệ support dataonline.vn
